# Scotland-EURO-2024
Using match event data, a visual representation of Scotland´s three group stage matches during the 2024 EURO.
The focal point of this project was the defence-attack transition phase from the Scottish National Team throughout the tournament, highlighting metrics such as ball recovery zones, players with the most ball recoveries per zone, and main passing zones after recovery. 
This was done as a practical, in-class activity as part of my Master´s in Big Data and Data Analysis in Football at Real Madrid Graduate School.
